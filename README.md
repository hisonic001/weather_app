# 실시간 날씨 앱
**Code Url**

https://github.com/hisonic001/weather_app



**About the Project**

실시간으로 지역의 날씨를 파악하여 사용자에게 제공하는 앱

구현하고 싶은 기능

- 접속하는 기기의 GPS 정보를 제공 동의 받아 자동으로 현재 지역 설정
- 제공된 정보를 바탕으로 weather API를 통해 날씨 정보 받기
- User에게 간단한 UI를 통해 지역, 날씨, 온도, 간단한 코멘트 전달
